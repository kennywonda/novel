# Background Music

Place your romantic background music here as `romantic-bg.mp3`.

**Where to get royalty-free romantic music:**
- [Pixabay Music](https://pixabay.com/music/) (100% free)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary/music)
- [Uppbeat](https://uppbeat.io/)

Search for: "romantic piano", "soft instrumental", "cinematic love", "fairy tale music"

**File requirements:**
- Format: MP3
- Name: `romantic-bg.mp3`
- Duration: 2-5 minutes (will loop automatically)
- Volume: Keep it subtle — it's background ambiance

If you don't add music, the music player button won't appear. The app will still work fine without it.
